# Тест-кейсы – UI Automation Test Store
## 1. Проверка сортировки товаров в категории Men

### TК-01: Сортировка по имени A → Z
**Шаги:**
1. Открыть главную страницу
2. Перейти в категорию "Men"
3. В списке сортировки выбрать "Name A-Z"

**Ожидаемый результат:**
- Список товаров отсортирован по имени - по алфавиту, от начала (A → Z)

### TК-02: Сортировка по имени Z → A
**Шаги:**
1. Открыть главную страницу
2. Перейти в категорию "Men"
3. В списке сортировки выбрать "Name Z-A"

**Ожидаемый результат:**
- Список товаров отсортирован по имени - по алфавиту, с конца (Z → A)

### TК-03: Сортировка по цене (низкая → высокая)
**Шаги:**
1. Открыть главную страницу
2. Перейти в категорию "Men"
3. В списке сортировки выбрать "Price Low>High"

**Ожидаемый результат:**
- Список товаров отсортирован по цене - по возрастанию

### TК-04: Сортировка по цене (высокая → низкая)
**Шаги:**
1. Открыть главную страницу
2. Перейти в категорию "Men"
3. В списке сортировки выбрать "Price High>Low"

**Ожидаемый результат:**
- Список товаров отсортирован по цене - по убыванию

## 2. Проверка поисковой выдачи и корзины

### ТК-05: Добавление товаров из поиска и проверка итоговой суммы корзины

**Шаги:**
1. Открыть главную страницу
2. Ввести в поле поиска значение "shirt"
3. Выполнить поиск
4. В списке сортировки выбрать "Name A-Z"
5. Открыть второй товар из результатов поиска
6. Указать случайное количество товара
7. Добавить товар в корзину
8. Повторно выполнить поиск по значению "shirt"
9. В списке сортировки выбрать "Name A-Z"
10. Открыть третий товар из результатов поиска
11. Указать случайное количество товара
12. Добавить товар в корзину
13. Перейти в корзину
14. Найти товар с наименьшей ценой за единицу
15. Увеличить количество найденного товара в 2 раза
16. Обновить корзину
17. Проверить конечную общую цену товаров

**Ожидаемый результат:**
- В корзине отображаются 2 добавленных товара
- Для самого дешёвого товара количество увеличено в 2 раза
- Итоговая сумма корзины пересчитана корректно и соответствует сумме всех товаров с учетом обновленного количества
